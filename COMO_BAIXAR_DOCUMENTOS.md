# 📥 COMO BAIXAR E IMPRIMIR DOCUMENTOS GMUD

## 🎯 Objetivo

Quando você cria uma manutenção no sistema GMUD, agora pode gerar um documento Word profissional para:
- ✅ Imprimir e coletar assinaturas
- ✅ Sharepoint com gestores
- ✅ Manter arquivo para auditoria
- ✅ Usar como comprovante

---

## 🚀 PASSO A PASSO

### 1️⃣ Agendar uma Manutenção

**Ir para a aba "➕ Agendar"**

Preencher:
- 📅 Data da Manutenção
- ⚙️ Equipamento (ex: Compressor A)
- 📝 Descrição (o que será feito)
- 💡 Justificativa (por quê fazer)
- ⚠️ Risco (Baixo / Médio / Alto)

Clicar em **✅ Agendar GMUD**

### 2️⃣ Ir para "📋 Manutenções"

Ver tabela com todas as manutenções agendadas/em progresso/concluídas

### 3️⃣ Localizar Sua Manutenção

Usar filtros se necessário:
- 🔍 Digitar nome do equipamento
- Filter por Status

### 4️⃣ Clicar em "📄" (Download)

Na coluna de ações, há 3 botões:
- **📄** ← Click aqui para baixar documento
- **✏️** - Editar status
- **🗑️** - Deletar

### 5️⃣ Arquivo Baixa Automaticamente

Seu navegador baixa: `GMUD_42_Compressor_A.docx`

**Local de salvamento**:
- Windows: `Downloads` (geralmente)
- Mac: `Downloads`
- Linux: Depende de sua configuração

### 6️⃣ Abrir em Word

1. Ir para "Downloads"
2. Clicar 2x no arquivo
3. Abre em Microsoft Word (ou Google Docs, LibreOffice, etc)

### 7️⃣ Revisar Documento

Documento contém:
- [x] Data e hora da manutenção
- [x] Equipamento
- [x] Descrição completa
- [x] Justificativa
- [x] Nível de risco
- [x] Status
- [x] **Linhas para assinatura**

### 8️⃣ Imprimir ou Enviar

**Opção 1: Imprimir**
- Ctrl+P (Windows/Linux) ou Cmd+P (Mac)
- Escolher impressora
- Clicar "Imprimir"

**Opção 2: Enviar por Email**
- Anexar arquivo
- Enviar para gestor/responsável

**Opção 3: Salvar como PDF**
- Arquivo > Salvar Como
- Formato: PDF
- Salvar

## ❌ PROBLEMAS COMUNS

### "O documento não baixa"
**Solução**:
1. Verificar se popup foi bloqueado (clique ⬆️ na URL)
2. Permitir downloads para o site
3. Tentar novamente
4. Se persistir, avisar administrador

### "Arquivo abre corrompido"
**Solução**:
1. Fechar Word completamente
2. Deletar arquivo baixado
3. Baixar novamente
4. Se continuar, avisar administrador

### "Não consigo editar documento"
**Solução**:
1. Ver se está no modo de leitura (Word)
2. Clicar "Editar Documento"
3. Se está protegido, contate administrador
4. Ou fazer Copiar/Colar para novo documento

### "Impressora não funciona"
**Solução**:
1. Isso é problema do seu computador, não do sistema
2. Verificar drivers da impressora
3. Testar impressão com outro documento
4. Chamar suporte de TI

## 🎯 CASOS DE USO

### Caso 1: Manutenção Preventiva Agendada
1. Criar GMUD
2. Baixar documento
3. Entregar para técnico
4. Técnico executa
5. Assina documento
6. Arquivo guardado para auditoria

### Caso 2: Manutenção Corretiva (Emergência)
1. Criar GMUD (marcar como "Em Progresso")
2. Fazer manutenção
3. Depois que terminar, baixar documento
4. Imprimir e coletar assinatura
5. Guardar para registro

### Caso 3: Relatório para Gestor
1. Gerar vários documentos
2. Montar em pasta
3. Enviar por email em lote
4. Gestor revisa e aprova
