📌 INSTRUÇÕES PARA ADICIONAR O LOGO

Para adicionar o logo da sua empresa no cabeçalho do sistema GMUD:

1. Salve a imagem do logo com o nome: logo.png

2. Coloque o arquivo neste diretório:
   t:\TECNOLOGIA\SUPORTE\TECNICOS\HIGOR\Calemdario GMUD\static\images\

3. Recarregue a página no navegador (F5 ou Ctrl+F5)

4. O logo aparecerá no topo da página, acima do título "GMUD"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✨ DICAS:

✓ Recomendação de tamanho: 150x150px ou qualquer tamanho quadrado
✓ Formatos aceitos: PNG, JPG, SVG, GIF
✓ Para melhor resultado: use fundo transparente (PNG)
✓ Tamanho máximo: 500KB (será redimensionado automaticamente para 120px)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📂 Localização Exata:

static/
└── images/
    └── logo.png  ← Coloque aqui

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔧 Troubleshooting:

❌ Logo não aparece?
✓ Verificar se o arquivo se chama exatamente "logo.png"
✓ Verificar se está na pasta "images"
✓ Tentar atualizar página (Ctrl+F5)
✓ Verificar console do navegador (F12) por erros

❌ Logo muito grande/pequeno?
✓ Editar arquivo: static/images/logo.png com tamanho diferente
✓ Ou editar CSS em static/index.html
   Procurar por: .header-logo { max-width: 120px; }
   Mudar 120px para outro valor
